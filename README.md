# GeoGroups Demo

Одностраничное веб-приложение для работы с геоточками и личной перепиской между пользователями.

## Возможности

- Интерактивная карта (Leaflet + OpenStreetMap).
- Добавление геоточек по клику на карту.
- Редактирование и удаление **только своих** точек.
- Фильтры по пользователю и по группам.
- Личный чат между пользователями.
- Данные сохраняются в `localStorage` браузера.

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
python3 -m http.server 4173
```

Далее откройте `http://localhost:4173`.
